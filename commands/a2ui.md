---
description: "Generates A2UI JSON from a natural language UI description. Usage: /a2ui [description of the UI to build]"
---

Use the `a2ui-builder` agent to generate A2UI JSON.

The text after `/a2ui` is the UI description. Pass it directly to the `a2ui-builder` agent.

If no description was provided, ask: "What UI would you like to generate? Describe the layout, sections, and key components."
